# WinVen - El gestor de ventanas PARA LOS WEBONES IGUAL QUE YO

WinVen es esa herramienta que ws un gestor de ventanas para Windows que te permite controlar todo con el teclado para optimizar tu experiencia en la PC.

## Atajos de teclado (los que te salvan la vida)

### Configuracion y Modo Juego
- Ctrl + Alt + 0: Abre el panel de configuracion donde podes configurar atajos de ejecucion de programas, layouts con la tecla AltGr + numero y demas que decidas y poner si quieres que se ejecute al iniciar windows.

## - Ctrl + Alt + J: Activa el Modo Juego que desactiva los atajos de movimiento y redimensionado y activa un cartelito arriba a la izquierda que dice JUEGO para que sepas que esta prendido.

### Movimientol de ventanas (Ctrl + WASD)
- Ctrl + W: La ventana se mueve hacia arriba.
- Ctrl + A: La ventana se mueve a la izquierda.
- Ctrl + D: La ventana se mueve a la derecha.
- Ctrl + Shift + S: La ventana se mueve para abajo. Esta es asi ys que me jodia con el VS code al guardar un documento.

### Redimensionado (Alt + WASD)
- Alt + W: Achica la ventana desde abajo hacia arriba.
- Alt + S: Agranda la ventana desde abajo.
- Alt + A: Achica la ventana desde la derecha hacia la izquierda.
- Alt + D: Agranda la ventana hacia la derecha.
- Alt + Shift + W: Agranda la ventana hacia arriba.
- Alt + Shift + S: Achica la ventana desde arriba hacia abajo.
- Alt + Shift + A: Agranda la ventana hacia la izquierda.
- Alt + Shift + D: Achica la ventana desde la izquierda hacia la derecha.

### Navegacion (Ctrl + Alt + Flechas)
- Ctrl + Alt + Izquierda: Pasas a la ventana anterior que tenias abierta.
- Ctrl + Alt + Derecha: Pasas a la siguiente ventana. Re util para saltar entre carpetas y el navegador.
- Ctrl + Alt + Arriba: Trae la ventana que tenes seleccionada bien al frente de todo.
- Ctrl + Alt + Abajo: Manda la ventana al fondo para que no moleste.

### Los botones magicos
- Ctrl + Alt + 1: Cicla la ventana por 25 posiciones fijas en la pantalla. Es ideal para acomodar todo rapido segun lo que estes haciendo.
- Ctrl + Alt + 2: Si moviste una ventana y te arrepentiste pulsa esto y vuelve a donde estaba antes de que la tocaras.
- Ctrl + Alt + 3: Ordena todas las ventanas que tenes abiertas para que ninguna tape a la otra. Magia pura.
- Ctrl + Shift + D: Cierra la ventana activa de una pero de forma segura para que no se rompa nada.
- Ctrl + Shift + T: Activa o desactiva la transparencia. Tremendo si queres ver que hay detras sin minimizar.

## Funciones Especiales

### El Modo Juego es clave
Cuando activas el Modo Juego con Ctrl + Alt + J pasan estas cosas:
- Se bloquean los atajos de movimiento y redimensionado para que no muevas una ventana sin querer mientras jugas.
- Aparece un cartelito arriba a la izquierda que dice JUEGO para que sepas que esta prendido.
- Se desactivan las cosas que consumen CPU para que te vaya a mas FPS y no tengas lag.

### Atajos personalizados
En el panel podes configurar tus propios atajos de teclado:
- Layouts: Podes crear tus propias formas de acomodar ventanas y ponerles una tecla con Ctrl + Alt.
- Apps: Podes poner atajos (normalmente con AltGr) para abrir programas al toque como el VS Code o el navegador sin buscarlos en el menu de inicio.

### Cosas que puedes configurar en el Panel
Si entras a la config (Ctrl + Alt + 0) tenes un par de opciones:
- Margen: Podes elegir que tan pegadas quedan las ventanas cuando se ordenan.
- Transparencia: Manejas que tan "fantasma" se vuelven las ventanas.
- Sonidos: Si te molesta el ruidito de los atajos lo podes sacar de ahi.
- Animaciones: Para que las ventanas se muevan suave o de una.
- Inicio automatico: Para que el programa arranque solo cuando prendes la compu y no tengas que buscarlo.

## Instalacion y como se usa
1. Abres el gestor_ven.exe y listo.
2. El programa se queda trabajando en las sombras para no molestarte.
3. Si queres cambiar algo dale a Ctrl + Alt + 0 y ahi tenes todos los botones y sliders para dejarlo como mas te guste.
